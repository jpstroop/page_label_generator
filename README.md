# Page Label Generator

Study to implement https://github.com/pulibrary/plum/issues/7

## Requires

Node >= 4.2.2

## Install

```
$ npm install
```

You might also want to install babel-cli globally:

```
$ npm install --global babel-cli
```

## Test

```
$ npm test
```

## Transpile to es5

```
babel src --out-dir lib
```
